## Обзор

**RedX Game Library** — это ультрасовременное приложение для управления играми, созданное для того, чтобы сделать игры доступными для всех. Оно обеспечивает бесшовный способ организации, управления и улучшения вашего игрового опыта в Steam без использования сторонних сервисов.

> **Доступные языки:** [RU](README.ru.md) | [EN](README.md) | [TR](README.tr.md) | [DE](README.de.md) | [JP](README.ja.md) | [CN](README.zh.md)

## Функции

### [>] Управление игровой библиотекой
- **Просмотр и поиск:** Исследуйте тысячи игр с интуитивным интерфейсом
- **Установленные игры:** Отслеживайте все добавленные игры в одном месте
- **Детали игры:** Просматривайте подробную информацию о любой игре

### [o] Онлайн-функции
- **Онлайн-фиксы:** Доступ к онлайн-фиксам для поддерживаемых игр
- **Обходные решения:** Быстрый доступ к обходным опциям для игр
- **Удаленная установка:** Добавляйте игры в библиотеку из любого места через веб-интерфейс

### [#] Безопасность и доверие
- **Безопасно:** Нет сторонних зависимостей для установки игр
- **Прямая интеграция:** Работает напрямую с вашей существующей установкой Steam
- **Проверенные источники:** Все фиксы и обходы поступают из проверенных источников

### [O] Современный интерфейс
- **Темная тема:** Элегантный красно-черный дизайн
- **Плавные анимации:** 60 FPS анимированные экраны загрузки и переходы
- **Настраиваемый:** Регулируемые настройки для игр на странице, цензуры и др.

### [~] Фоновые функции
- **Системный трей:** Работает в фоне с интеграцией в системный трей
- **Авто-опрос:** Автоматически проверяет запросы на загрузку
- **Авто-обновления:** Встроенная система обновлений поддерживает актуальность

## Установка

### Требования
- Операционная система Windows
- Установленный Steam на вашей системе
- Интернет-соединение для начальной настройки

### Шаги
1. Загрузите последний релиз со страницы [Releases](https://github.com/Scriptez1/RedX/releases)
2. Распакуйте архив в предпочитаемое место
3. Запустите `RedX.exe` или `main.py` (если запускаете из исходников)
4. Следуйте инструкциям по аутентификации при первом запуске

### Методы аутентификации
Выберите предпочтительный метод аутентификации при первом запуске:
- **GitHub + YouTube:** Поставьте звезду репозиторию и подпишитесь на канал
- **Discord:** Присоединитесь к нашему серверу и используйте команду `/key` для 24-часового ключа доступа

## Использование

### Начало работы
1. **Запустите** приложение
2. **Аутентифицируйтесь** используя предпочтительный метод
3. **Установите путь Steam** если не обнаружен автоматически
4. **Просмотрите** библиотеку игр и начните добавлять игры!

### Основные разделы

| Раздел | Описание |
|---------|-------------|
| **Главная** | Панель с статистикой и быстрым доступом |
| **Все игры** | Просмотр полной базы данных игр |
| **Установленные** | Управление добавленными играми |
| **Онлайн-фиксы** | Загрузка мультиплеерных фиксов |
| **Обход** | Доступ к обходным решениям |

### Быстрые действия
- **Запросить игру:** Не нашли игру? Запросите ее добавление
- **Жалоба/Предложение:** Отправьте отзыв напрямую разработчикам
- **Перезапуск Steam:** Кнопка для быстрого перезапуска Steam

## Языки

RedX Game Library поддерживает несколько языков:

- [RU] **Русский** - Полная поддержка интерфейса и игрового контента
- [EN] **English** - Полная поддержка интерфейса и игрового контента
- [TR] **Türkçe** - Полный турецкий перевод с описаниями игр
- [DE] **Deutsch** - Немецкая поддержка интерфейса и контента
- [JP] **日本語** - Японская поддержка интерфейса и контента
- [CN] **中文** - Китайская поддержка интерфейса и контента

Язык можно изменить в Настройках. Описания игр автоматически переводятся в зависимости от выбранного языка.
