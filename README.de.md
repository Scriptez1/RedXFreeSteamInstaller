## Übersicht

**RedX Game Library** ist eine ultra-moderne Spielverwaltungsanwendung, die entwickelt wurde, um Gaming für alle zugänglich zu machen. Sie bietet eine nahtlose Möglichkeit, Ihr Steam-Spieleerlebnis zu organisieren, zu verwalten und zu verbessern, ohne sich auf Drittanbieter-Dienste zu verlassen.

> **Verfügbare Sprachen:** [DE](README.de.md) | [EN](README.md) | [TR](README.tr.md) | [RU](README.ru.md) | [JP](README.ja.md) | [CN](README.zh.md)

## Funktionen

### [>] Spielebibliothek-Verwaltung
- **Durchsuchen & Suchen:** Erkunden Sie Tausende von Spielen mit einer intuitiven Oberfläche
- **Installierte Spiele:** Behalten Sie alle Ihre hinzugefügten Spiele an einem Ort im Blick
- **Spieldetails:** Zeigen Sie umfassende Informationen über jedes Spiel an

### [o] Online-Funktionen
- **Online-Fixes:** Zugriff auf Online-Multiplayer-Fixes für unterstützte Spiele
- **Bypass-Lösungen:** Schneller Zugriff auf Spiel-Bypass-Optionen
- **Remote-Installation:** Fügen Sie Spiele von überall über die Weboberfläche zu Ihrer Bibliothek hinzu

### [#] Sicherheit & Vertrauen
- **Sicher:** Keine Drittanbieter-Abhängigkeiten für die Spielinstallation
- **Direkte Integration:** Funktioniert direkt mit Ihrer bestehenden Steam-Installation
- **Vertrauenswürdige Quellen:** Alle Fixes und Bypasses stammen aus verifizierten Quellen

### [O] Moderne Oberfläche
- **Dunkles Thema:** Elegant rot-schwarzes Design
- **Flüssige Animationen:** 60 FPS animierte Ladebildschirme und Übergänge
- **Anpassbar:** Einstellbare Optionen für Spiele pro Seite, Zensur und mehr

### [~] Hintergrundfunktionen
- **System-Tray:** Läuft im Hintergrund mit System-Tray-Integration
- **Auto-Polling:** Überprüft automatisch auf Download-Anfragen
- **Auto-Updates:** Integriertes Aktualisierungssystem hält Sie auf dem neuesten Stand

## Installation

### Voraussetzungen
- Windows-Betriebssystem
- Auf Ihrem System installiertes Steam
- Internetverbindung für die Ersteinrichtung

### Schritte
1. Laden Sie die neueste Version von der [Releases](https://github.com/Scriptez1/RedX/releases)-Seite herunter
2. Entpacken Sie das Archiv an Ihrem bevorzugten Ort
3. Führen Sie `RedXGameLibrary.exe`
4. Folgen Sie den Authentifizierungsaufforderungen beim ersten Start

### Authentifizierungsmethoden
Wählen Sie Ihre bevorzugte Authentifizierungsmethode beim ersten Start:
- **GitHub + YouTube:** Sternen Sie das Repository und abonnieren Sie den Kanal
- **Discord:** Treten Sie unserem Server bei und verwenden Sie den `/key`-Befehl für einen 24-Stunden-Zugriffsschlüssel

## Benutzung

### Erste Schritte
1. **Starten** Sie die Anwendung
2. **Authentifizieren** Sie sich mit Ihrer bevorzugten Methode
3. **Setzen Sie den Steam-Pfad**, falls nicht automatisch erkannt
4. **Stöbern** Sie in der Spielebibliothek und fangen Sie an, Spiele hinzuzufügen!

### Hauptbereiche

| Bereich | Beschreibung |
|---------|-------------|
| **Startseite** | Dashboard mit Statistiken und Schnellzugriff |
| **Alle Spiele** | Durchsuchen Sie die komplette Spieldatenbank |
| **Installiert** | Verwalten Sie Ihre hinzugefügten Spiele |
| **Online-Fixes** | Laden Sie Multiplayer-Fixes herunter |
| **Bypass** | Zugriff auf Bypass-Lösungen |

### Schnellaktionen
- **Spiel anfragen:** Spiel nicht gefunden? Fordern Sie es an, hinzugefügt zu werden
- **Beschwerde/Vorschlag:** Senden Sie Feedback direkt an die Entwickler
- **Steam-Neustart:** Schnelle Schaltfläche zum Neustarten von Steam bei Bedarf

## Sprachen

RedX Game Library unterstützt mehrere Sprachen:

- [DE] **Deutsch** - Deutsche Oberflächen- und Inhaltsunterstützung
- [EN] **English** - Vollständige Oberflächen- und Spielinhaltsunterstützung
- [TR] **Türkisch** - Vollständige türkische Übersetzung mit Spielbeschreibungen
- [RU] **Russisch** - Russische Oberflächen- und Inhaltsunterstützung
- [JP] **Japanisch** - Japanische Oberflächen- und Inhaltsunterstützung
- [CN] **Chinesisch** - Chinesische Oberflächen- und Inhaltsunterstützung

Die Sprache kann in den Einstellungen geändert werden. Spielbeschreibungen werden basierend auf Ihrer ausgewählten Sprache automatisch übersetzt.
