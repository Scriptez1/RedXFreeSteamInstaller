## 概述

**RedX Game Library** 是一款超现代的游戏管理应用程序，旨在让每个人都能享受游戏。它提供了一种无缝的方式来组织、管理和增强您的Steam游戏体验，而无需依赖第三方服务。

> **可用语言：** [CN](README.zh.md) | [EN](README.md) | [TR](README.tr.md) | [DE](README.de.md) | [RU](README.ru.md) | [JP](README.ja.md)

## 功能

### [>] 游戏库管理
- **浏览与搜索：** 通过直观的界面探索数千款游戏
- **已安装游戏：** 在一个地方跟踪所有添加的游戏
- **游戏详情：** 查看任何游戏的全面信息

### [o] 在线功能
- **在线修复：** 访问支持游戏的在线多人修复
- **绕过解决方案：** 快速访问游戏绕过选项
- **远程安装：** 通过网页界面从任何地方将游戏添加到您的库中

### [#] 安全与信任
- **安全：** 游戏安装无需第三方依赖
- **直接集成：** 直接与现有Steam安装配合使用
- **可信来源：** 所有修复和绕过均来自经过验证的来源

### [O] 现代界面
- **深色主题：** 时尚的红色和黑色设计
- **流畅动画：** 60 FPS动画加载屏幕和过渡效果
- **可定制：** 每页游戏数量、审查选项等的可调设置

### [~] 后台功能
- **系统托盘：** 通过系统托盘集成在后台运行
- **自动轮询：** 自动检查下载请求
- **自动更新：** 内置更新系统保持最新版本

## 安装

### 先决条件
- Windows操作系统
- 系统上已安装Steam
- 初始设置需要互联网连接

### 步骤
1. 从 [Releases](https://github.com/Scriptez1/RedX/releases) 页面下载最新版本
2. 将压缩包解压到您喜欢的位置
3. 运行 `RedX.exe` 或 `main.py`（如果从源代码运行）
4. 首次启动时按照认证提示操作

### 认证方法
首次运行时选择您首选的认证方式：
- **GitHub + YouTube：** 给仓库点赞并订阅频道
- **Discord：** 加入我们的服务器并使用 `/key` 命令获取24小时访问密钥

## 使用

### 开始使用
1. **启动** 应用程序
2. **使用您首选的方法进行认证**
3. **设置Steam路径**（如果未自动检测）
4. **浏览** 游戏库并开始添加游戏！

### 主要部分

| 部分 | 描述 |
|---------|-------------|
| **主页** | 带统计信息和快速访问的仪表板 |
| **所有游戏** | 浏览完整的游戏数据库 |
| **已安装** | 管理您添加的游戏 |
| **在线修复** | 下载多人游戏修复 |
| **绕过** | 访问绕过解决方案 |

### 快速操作
- **请求游戏：** 找不到游戏？请求添加
- **投诉/建议：** 直接向开发者发送反馈
- **重启Steam：** 需要时快速重启Steam的按钮

## 语言

RedX Game Library 支持多种语言：

- [CN] **中文** - 完整的界面和游戏内容支持
- [EN] **English** - 完整的界面和游戏内容支持
- [TR] **Türkçe** - 带游戏描述的完整土耳其语翻译
- [DE] **Deutsch** - 德语界面和内容支持
- [RU] **Русский** - 俄语界面和内容支持
- [JP] **日本語** - 日语界面和内容支持

语言可以在设置中更改。游戏描述会根据您选择的语言自动翻译。
