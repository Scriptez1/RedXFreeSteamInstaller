## 概要

**RedX Game Library**は、誰もがゲームを楽しめるように設計された超近代的なゲーム管理アプリケーションです。サードパーティのサービスに頼ることなく、Steamゲーム体験を整理、管理、向上させるためのシームレスな方法を提供します。

> **利用可能な言語:** [JP](README.ja.md) | [EN](README.md) | [TR](README.tr.md) | [DE](README.de.md) | [RU](README.ru.md) | [CN](README.zh.md)

## 機能

### [>] ゲームライブラリ管理
- **ブラウズ＆検索:** 直感的なインターフェースで数千のゲームを探索
- **インストール済みゲーム:** 追加したすべてのゲームを一か所で管理
- **ゲーム詳細:** 任意のゲームに関する包括的な情報を表示

### [o] オンライン機能
- **オンラインフィックス:** 対応ゲームのオンラインマルチプレイヤーフィックスにアクセス
- **バイパスソリューション:** ゲームバイパスオプションへのクイックアクセス
- **リモートインストール:** Webインターフェースを介してどこからでもライブラリにゲームを追加

### [#] セキュリティと信頼性
- **安全:** ゲームインストールにサードパーティの依存関係なし
- **直接統合:** 既存のSteamインストールと直接連携
- **信頼できるソース:** すべてのフィックスとバイパスは検証済みのソースから提供

### [O] モダンインターフェース
- **ダークテーマ:** 洗練された赤と黒のデザイン
- **スムーズなアニメーション:** 60 FPSのアニメーション付き読み込み画面とトランジション
- **カスタマイズ可能:** ページあたりのゲーム数、検閲オプションなどの調整可能な設定

### [~] バックグラウンド機能
- **システムトレイ:** システムトレイ統合でバックグラウンド実行
- **自動ポーリング:** ダウンロードリクエストを自動的にチェック
- **自動アップデート:** 最新バージョンを維持する組み込みアップデートシステム

## インストール

### 前提条件
- Windows OS
- システムにSteamがインストールされていること
- 初期セットアップ用のインターネット接続

### 手順
1. [Releases](https://github.com/Scriptez1/RedX/releases)ページから最新リリースをダウンロード
2. 好みの場所にアーカイブを展開
3. `RedXGameLibrary.exe` を実行
4. 初回起動時の認証プロンプトに従う

### 認証方法
初回実行時に好みの認証方法を選択：
- **GitHub + YouTube:** リポジトリにスターを付け、チャンネルを購読
- **Discord:** サーバーに参加し、`/key`コマンドで24時間アクセスキーを取得

## 使い方

### はじめに
1. **アプリケーションを起動**
2. **好みの方法で認証**
3. **Steamパスを設定**（自動検出されない場合）
4. **ゲームライブラリを閲覧**し、ゲームの追加を開始！

### 主要セクション

| セクション | 説明 |
|---------|-------------|
| **ホーム** | 統計情報とクイックアクセスのダッシュボード |
| **すべてのゲーム** | ゲームデータベース全体を閲覧 |
| **インストール済み** | 追加したゲームを管理 |
| **オンラインフィックス** | マルチプレイヤーフィックスをダウンロード |
| **バイパス** | バイパスソリューションにアクセス |

### クイックアクション
- **ゲームをリクエスト:** ゲームが見つからない？追加をリクエスト
- **苦情/提案:** 開発者に直接フィードバックを送信
- **Steam再起動:** 必要時にSteamを素早く再起動するボタン

## 言語

RedX Game Libraryは複数の言語をサポート：

- [JP] **日本語** - インターフェースとゲームコンテンツの完全なサポート
- [EN] **English** - インターフェースとゲームコンテンツの完全なサポート
- [TR] **Türkçe** - ゲーム説明付きの完全なトルコ語翻訳
- [DE] **Deutsch** - ドイツ語のインターフェースとコンテンツサポート
- [RU] **Русский** - ロシア語のインターフェースとコンテンツサポート
- [CN] **中文** - 中国語のインターフェースとコンテンツサポート

言語は設定から変更できます。ゲーム説明は選択した言語に基づいて自動的に翻訳されます。
