## ✨ 功能

### 🎨 现代界面
- **红黑主题**：优雅且现代的外观
- **响应式设计**：适应不同屏幕尺寸
- **动画元素**：平滑过渡和特效

### 🌍 多语言支持
- 🇹🇷 **土耳其语**（主要语言）
- 🇺🇸 **英语**
- 🇩🇪 **德语**
- 🇷🇺 **俄语**
- 🇯🇵 **日语**

### 🎮 游戏管理
- **游戏搜索**：快速高效的搜索系统
- **按类别筛选**：按游戏类别筛选
- **安装追踪**：查看和管理已安装的游戏
- **Steam 集成**：直接访问 Steam 商店页面

### 🔧 高级功能
- **自动更新**：通过 GitHub 自动检查更新
- **Steam 路径检测**：自动检测 Steam 文件夹
- **DLC 支持**：管理游戏 DLC
- **进度跟踪**：下载和安装进度查看

## 🚀 安装

1. 从 [Releases](https://github.com/Scriptez1/RedXFreeSteamInstaller/releases/latest) 下载最新的 `RedXGameLibrary.exe`
2. 双击 `RedXGameLibrary.exe` 运行
3. 无需安装！

## 📖 使用方法

### 首次启动
1. 打开应用时会自动检测 Steam 路径
2. 如果未检测到，可手动选择 Steam 文件夹
3. 设置页面显示的游戏数量（推荐：9）

### 游戏搜索与筛选
- **搜索框**：按游戏名称搜索
- **类别筛选**：从右侧面板选择类别
- **应用筛选**：点击“筛选”按钮应用选择

### 游戏管理
- **添加游戏**：点击游戏卡片上的“添加”按钮
- **删除游戏**：对已安装游戏使用“删除”按钮
- **在 Steam 查看**：打开游戏的 Steam 页面

### 切换语言
- 点击右侧面板的国旗图标切换语言
- 切换后应用会自动重启

### 最低要求

- **存储空间**：100 MB 可用空间
- **网络**：需要网络连接

- **内存**：8 GB
